This visualization is based on glyph chart.Here Y axis is the discovery year and X axis is the sequence number of that planet in that year.The glyph (Markers) consist of 3 types based on planet type(Gas Gaint, Neptune like and super earth).Connecting line between markers represents the channel. Channel is defined if the detection method used for the discovery of the planet is same or not.The line will be a connecting line(Channel) if the detection method is same for the adjoining planets else there will be no line between the adjoining planets.
